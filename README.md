# To-Do List Application

## Features
- **Add Tasks**: Users can add new tasks to their to-do list.
- **Edit Tasks**: Existing tasks can be modified or updated as needed.
- **Delete Tasks**: Users have the option to remove tasks from their list.
- **Mark as Completed**: Tasks can be marked as completed to help track progress.
- **Filter Tasks**: Users can filter tasks based on their completion status.
- **Due Dates**: Add due dates to tasks to ensure timely completion.

## Usage
1. **Installation**: Clone the repository to your local machine using `git clone <repository_url>`.
2. **Run the Application**: Open a terminal in the project directory and run the application using the command `python app.py`.
3. **Interface**: Access the application through your browser at `http://localhost:5000`.
4. **User Interaction**: Utilize the UI to add, edit, delete, or complete tasks as needed.
